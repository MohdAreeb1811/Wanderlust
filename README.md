# WanderLust

A full-stack travel listing web application inspired by Airbnb, built using Node.js, Express.js, MongoDB, Mongoose, and EJS.

## Features

- View all travel listings
- Add new listings
- Edit existing listings
- Delete listings
- MongoDB Atlas integration
- RESTful routing
- Responsive UI

## Tech Stack

### Frontend
- HTML
- CSS
- Bootstrap
- EJS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

## Installation

Clone the repository:

```bash
git clone https://github.com/MohdAreeb1811/Wanderlust.git
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
MONGO_URL=your_mongodb_connection_string
```

Start the server:

```bash
npm start
```

Open:

```text
http://localhost:8080/listings
```
